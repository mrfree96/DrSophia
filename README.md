# DrSophia
 
Dr Sophia can diagnose up to 10 most common skin disease.

Usage Hints:
- write the symptoms of skin disease to get a diagnosis.

## Supported Diseases:
- Acne
- Eczema
- Psoriasis
- Rosacea
- Hives (Urticaria)
- Dermatitis
- Skin Cancer
- Vitiligo
- Impetigo
- Shingles (Herpes Zoster)
